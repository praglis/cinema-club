W tym folderze będą umieszczane skrypty migracyjne
Format:
V{rok}_{miesiac}_{dzien}_{godzina}_{minuta}_{sekunda}__CC_{TASK_ID}_{NAZWA}.sql

Przykładowa nazwa pliku:
V2019_10_27_08_09_00__CC_10_INIT_SCRIPTS_001.sql

1. Gdy zostanie dodany skrypt, który okaże się błędyny:
Należy stworzyć nowy skrypt z późniejszą datą, który poprawia błędy w bazie danych
Nie należy ponownie edytować skryptów dodanych w przeszłości!
